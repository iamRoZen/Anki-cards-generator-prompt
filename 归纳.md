# 归纳
Make an outline/a summary in MD format.
Satisfying : professional, specific, detailed, easy to understand, and recapitulative.
## Settings
- Format: .txt (UTF-8)
- Language: Chinese
- Range: the text provided

## Step
1. Summarize the `title` 
2. Find `topic`s for each section. If there is no specific section, split the subtopic independently. Find `subtopic`s as well.
3. Find important points coated with `**` or `*`. Use iteral list to highlight these points. If there is no point, dismiss this step.
4. Conclude the `Content` which shown in the front of the text. is the subtitles and lower structure, use 1/1.1/1.1.1 system to mark different topics.

## Note
1. You can use 4/5 grades of subtopic, like #### or ##### instead of complex list, but use normal text when there is no more content below.
2. Use order list instead of unordered list in the content.
3. Ensure the output is based on the provided text.
4. Maintain the `[[]]` links and `![[Pasted image]]` picture links, and put them in proper place.
5. Demonstrate all complex `lists` in the form of multilayer titles.

## Don'ts
1. Stop the content / list with no dots or commas.
2. Do not use `---` as a split or any other split.
